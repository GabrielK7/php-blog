# Blog Web Application

Simple blog application built with PHP.

## Features

- Public blog with articles
- Article detail page
- Login / logout
- Admin section with full post management (CRUD)
- Responsive layout
- Custom routing system (self-built Router)

## Technologies

- PHP
- MySQL (PDO)
- HTML / CSS
